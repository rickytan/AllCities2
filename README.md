AllCities2
==========

精简一点的中国城市地名数据

使用方法
---
在 `addr.json` 中有两个主要字段：`link`，`def`。其中 `link` 分层级分别表示**省**，**市**，**县/区**，没有更详细的了（更详细请转到<https://github.com/rickytan/AllCities>），`def` 表示编号对应的中文名。如：

```json
{
  "link": {
    110000: {
      110000: [
        110101,
        110102
      ]
    },
    120000: {
      120000: [
        120101
      ]
    }
}
```

表示：

    北京（省）
     |`--北京（市）
     |    |`--东城区
     |     `--西城区
    天津（省）
      `--天津（市）
           `--和平区
